# WAAMLABZ Dark Theme

## Visual Studio Code compatible (.vsix)

VScode theme based on WAAMLABZ' greenish-hued colour scheme.

## Installation

Follow these steps to install the **WAAMLABZ Dark** theme in your VS Code environment:

### 1. Download the Installer
* Go to the [Releases](https://github.com/EoinCaughey/waamlabz-dark/releases) page.
* Download the latest `.vsix` file (e.g., `waamlabz-dark-0.0.1.vsix`).

### 2. Install in VS Code
1. Open **Visual Studio Code**.
2. Open the **Extensions** view by clicking the square icon on the left sidebar or pressing `Ctrl+Shift+X`.
3. Click the **More Actions...** icon (the three horizontal dots `...`) in the top-right corner of the Extensions pane.
4. Select **Install from VSIX...** from the dropdown menu.
5. Locate and select the `.vsix` file you downloaded in Step 1.

### 3. Activate the Theme
1. Once installed, press `Ctrl+K` followed by `Ctrl+T` to open the Theme selection menu.
2. Select **WAAMLABZ Dark** from the list.
3. Enjoy your high-contrast, WAAMLABZ-themed environment!
